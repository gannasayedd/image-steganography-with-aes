AES + LSB Steganography

This local web interface wraps your Python steganography engine. It encrypts the message with AES, hides it inside a PNG using LSB embedding, and reports MSE, PSNR, and processing time.

Upload only PNG images for safe hidden-data storage.
Use the same key for encryption and decryption.
Download the generated stego image after encoding.

Encode Message

Hide an AES-encrypted secret inside a PNG cover image.

Decode Message

Extract the hidden payload and decrypt it with the correct key.

Run the server with .\.venv\Scripts\python.exe web_app.py, then open http://127.0.0.1:8000 in your browser.
